SUSTAINABLE CATALYST PLATFORM CORE v2.1.0
Knowledge Graph and Relationship Engine

Place in ~/Downloads:

sustainable-catalyst-platform-core-v2.1.0-repo.zip
push_platform_core_v2_1_0_to_github.sh

Run:

cd ~/Downloads
chmod +x push_platform_core_v2_1_0_to_github.sh
./push_platform_core_v2_1_0_to_github.sh

The script uses SSH for every Git operation. It contains no HTTPS fallback.

RENDER

Add or update:

SC_CORE_MAX_GRAPH_DEPTH=4
SC_CORE_EXPLORER_ENABLED=true

curl https://YOUR-PLATFORM-CORE.onrender.com/health
curl https://YOUR-PLATFORM-CORE.onrender.com/v1/predicates
curl "https://YOUR-PLATFORM-CORE.onrender.com/v1/graph/sc:product:research-librarian/neighborhood"
