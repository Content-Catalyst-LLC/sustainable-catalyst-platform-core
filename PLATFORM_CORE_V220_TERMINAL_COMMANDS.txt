Run exactly:

cd ~/Downloads
chmod +x PUSH_PLATFORM_CORE_V220_CLEAN.sh
./PUSH_PLATFORM_CORE_V220_CLEAN.sh

The banner must say:

PLATFORM CORE v2.2.0 — CLEAN EVIDENCE LEDGER PUSH

The script:

- Uses the existing GitHub CLI authentication
- Uses SSH for every Git operation
- Contains no HTTPS fallback
- Handles an existing or empty repository
- Safety-stashes unfinished local changes
- Deletes any damaged backend/.venv
- Excludes .venv from rsync
- Recreates the virtual environment from scratch
- Uses .venv/bin/python directly
- Runs all tests
- Runs a clean migration and seed smoke test
- Builds the WordPress plugin
- Commits and pushes main
