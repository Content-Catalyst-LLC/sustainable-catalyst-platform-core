SUSTAINABLE CATALYST PLATFORM CORE v2.4.0
Trust Center and Evaluation Framework

Place these files in ~/Downloads:

1. sustainable-catalyst-platform-core-v2.4.0-repo.zip
2. PUSH_PLATFORM_CORE_V240_FINAL.sh

Run exactly:

cd ~/Downloads
chmod +x PUSH_PLATFORM_CORE_V240_FINAL.sh
./PUSH_PLATFORM_CORE_V240_FINAL.sh

The banner must say:

PLATFORM CORE v2.4.0 — FINAL TRUST CENTER PUSH

The script:

- Uses existing GitHub CLI authentication
- Uses SSH for every Git operation
- Has no HTTPS or password fallback
- Handles an existing or empty repository
- Safety-stashes unfinished local changes
- Removes damaged virtual environments
- Excludes .venv from rsync
- Recreates the virtual environment from scratch
- Uses .venv/bin/python directly
- Runs all 36 tests
- Runs migration 0005 and seed smoke tests
- Runs the default Trust Center evaluation suite
- Builds the WordPress connector
- Copies Python and JavaScript SDK downloads
- Copies the Postman collection
- Commits and pushes main
